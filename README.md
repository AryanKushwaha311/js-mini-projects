# 🚀 JavaScript Mini Projects

A collection of beginner to intermediate JavaScript mini projects built to strengthen core programming concepts and problem-solving skills.

---

## 📂 Projects Included

* 🔢 **Counter Program**
  Simple counter with increment, decrement, and reset functionality.

* 🎲 **Roll the Dice**
  Generates random dice values (1–6) on each roll.

* 🎂 **Age Submission**
  Takes user input and validates age conditions.

* 🎯 **Number Guessing Game**
  Guess the correct number with feedback hints.

* 🌡️ **Temperature Conversion**
  Convert between Celsius and Fahrenheit.
